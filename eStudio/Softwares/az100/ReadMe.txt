            az100压缩解压缩PC工具使用说明

文件压缩
az100 e input output
input   压缩源文件路径，支持全路径和相对路径
output  压缩后目标文件路径，支持全路径和相对路径

文件解压
az100 d input output
input   解压源文件路径，支持全路径和相对路径
output  解压后目标文件路径，支持全路径和相对路径
